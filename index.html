Telepresence with Galileo

Golem app allows people to remotely control a smartphone
plugged into a Galileo device. All in three simple steps.

1. Plug into Galileo

Just plug it! Make sure the smartphone is supported by Galileo hardware.

2. Enable Host Mode

This will setup a PIN-protected room for remote devices to connect to over the Internet.

3. Use Remote Mode

Connect to host device by entering PIN code shared by the host. Use joystick to control Galileo.
